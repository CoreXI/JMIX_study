# Задание — PostgreSQL

## Задание: Создать и наполнить базу данных сотрудников

**Цель:** Написать SQL-запросы для работы с данными — те же операции, что делали через Stream API, но на уровне БД.

**Что нужно сделать:**

1. Установи PostgreSQL локально (или используй Docker из следующей темы)
2. Создай базу данных `jmix_study`
3. Создай таблицу:
   ```sql
   CREATE TABLE employee (
       id             BIGSERIAL PRIMARY KEY,
       name           VARCHAR(255) NOT NULL,
       department     VARCHAR(100),
       salary         NUMERIC(10, 2),
       experience     INTEGER
   );
   ```
4. Добавь 8-10 сотрудников через `INSERT`
5. Напиши запросы:
   - Все сотрудники из отдела IT
   - Средняя зарплата по компании
   - Количество сотрудников в каждом отделе (`GROUP BY`)
   - Самый высокооплачиваемый сотрудник с опытом > 3 лет
   - Имена сотрудников с зарплатой выше средней (через подзапрос)

**Задание выполнено, если:**
- [ ] Таблица создана и содержит 8+ записей
- [ ] Все 5 запросов написаны и возвращают корректные результаты
- [ ] Запрос с зарплатой выше средней использует подзапрос (`SELECT AVG(salary) FROM employee`)
- [ ] `GROUP BY` возвращает количество сотрудников по каждому отделу

**Подсказка:** Последний запрос выглядит так:
```sql
SELECT name FROM employee
WHERE salary > (SELECT AVG(salary) FROM employee);
```
Это SQL-аналог того, что ты делал через Stream API в блоке 06.
