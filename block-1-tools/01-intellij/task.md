# Задание — IntelliJ IDEA

## Задание: Освоить среду разработки

**Цель:** Научиться уверенно работать в IntelliJ IDEA — находить файлы, запускать код, использовать отладчик.

**Что нужно сделать:**

1. Открой проект `JMIX_study` в IntelliJ IDEA
2. Найди файл `Main.java` из блока 06-streams через `Shift+Shift` (поиск по файлам)
3. Запусти его через `Ctrl+R` / `Shift+F10` — убедись что вывод появляется в консоли
4. Поставь breakpoint на строку `main.setupDepartments()` и запусти в режиме отладки
5. Используй `Step Into` чтобы зайти внутрь метода и посмотреть как заполняется `staff`
6. Отформатируй любой файл через `Cmd+Option+L` / `Ctrl+Alt+L`
7. Переименуй локальную переменную в любом методе через `Shift+F6` — убедись что все вхождения обновились

**Задание выполнено, если:**
- [ ] Программа запущена и выводит результат в консоль IntelliJ
- [ ] Отладчик остановился на breakpoint и ты видел переменные в панели Variables
- [ ] Использован `Shift+Shift` для поиска файла
- [ ] Переименование через `Shift+F6` обновило все вхождения

**Подсказка:** В панели Variables отладчика можно раскрывать объекты — найди `staff` и посмотри на список сотрудников внутри.
