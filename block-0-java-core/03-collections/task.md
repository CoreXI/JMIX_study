# Задание 3 — Коллекции

**Цель:** Применить List, Map и Set в связанном сценарии.

---

## Что нужно сделать

Напиши программу — простой учёт студентов и их оценок.

### Шаг 1 — Список студентов

Создай `List<String>` с именами 5 студентов. Выведи их порядковые номера и имена:
```
1. Алиса
2. Боб
3. Карл
4. Диана
5. Ева
```

### Шаг 2 — Оценки

Создай `Map<String, Integer>` — имя студента → оценка (от 1 до 100). Заполни данными для всех 5 студентов.

Выведи всех студентов с оценками:
```
Алиса: 92
Боб: 78
...
```

### Шаг 3 — Уникальные оценки

Из Map создай `Set<Integer>` уникальных оценок (некоторые студенты могут иметь одинаковые — добавь пару дубликатов намеренно). Выведи количество уникальных оценок.

### Шаг 4 — Анализ

Найди и выведи:
- Имя студента с **максимальной** оценкой
- Среднюю оценку по группе (округли до одного знака)

---

## Задание выполнено, если:

- [ ] Использованы все три типа коллекций: `List`, `Map`, `Set`
- [ ] Set содержит только уникальные оценки (проверь что дубликаты не попали)
- [ ] Максимум и среднее вычисляются через перебор, не вручную
- [ ] Программа компилируется и выводит корректные результаты

---

## Подсказка

Для поиска максимума перебирай `Map.entrySet()`. Среднее: сумма значений / количество записей. Сумму удобно копить в переменной типа `double`.
