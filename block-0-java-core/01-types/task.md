# Задание 1 — Типы данных и переменные

**Цель:** Закрепить работу с примитивными типами, String, var и приведением типов.

---

## Что нужно сделать

Создай файл `Main.java` и напиши программу, которая:

1. Объявляет переменные для описания товара в интернет-магазине:
   - название товара (`String`)
   - цена (`double`)
   - количество на складе (`int`)
   - признак активности товара (`boolean`)
Создай файл man
2. Выводит в консоль строку вида:
   ```
   Товар: Ноутбук | Цена: 1299.99 | Склад: 5 шт | Активен: true
   ```

3. Считает итоговую стоимость всего товара на складе (`цена * количество`) и выводит:
   ```
   Итого на складе: 6499.95
   ```

4. Явно приводит `double`-цену к `int` (обрезает копейки) и выводит:
   ```
   Цена без копеек: 1299
   ```

5. Повторяет объявление хотя бы одной переменной через `var`.

---

## Задание выполнено, если:

- [ ] Все 4 переменные объявлены с правильными типами
- [ ] Программа компилируется и запускается без ошибок
- [ ] Вывод в консоль совпадает с ожидаемым форматом
- [ ] Использован `var` хотя бы один раз
- [ ] Явное приведение типа `(int)` присутствует в коде

---

## Подсказка

Для вывода в консоль используй `System.out.println()`.
Конкатенацию строк можно делать через `+` или через `String.format()` — второй вариант чище, но оба правильные.
