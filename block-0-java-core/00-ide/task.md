# Задание 0 — Настройка рабочей среды

## Задание: Первый запуск в IntelliJ IDEA

**Цель:** настроить IDE и убедиться, что всё работает до того, как начнём писать код.

**Что нужно сделать:**

1. Установить IntelliJ IDEA Community Edition
2. Создать новый Java-проект (`File → New Project → Java`)
3. Убедиться, что JDK выбран (версия 17 или 21)
4. В файле `Main.java` написать:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, JMIX study!");
    }
}
```

5. Запустить через зелёный треугольник рядом с `main()` или `Shift+F10`
6. Убедиться, что в панели **Run** внизу появился вывод

**Задание выполнено, если:**
- [ ] IDEA установлена и открывается
- [ ] Проект создан, JDK подключён
- [ ] В консоли Run появилась строка `Hello, JMIX study!`
- [ ] Ты знаешь, где находятся: дерево файлов, редактор, панель Run, терминал

**Подсказка:** если JDK не находится автоматически — в настройках проекта (`File → Project Structure → SDK`) можно указать путь вручную или скачать через IDEA.
